# 概要

このタスクでは大きなデータの中から特定のデータを見つけ出すためのロジックを二分探索を使って実装します。

- 二分探索を用いたロジックを実装する経験をしたい方

## なぜ必要なのか

二分探索はソートされたリストや配列から特定の要素を探すためのアルゴリズムであり、線形探索と比べて計算量の観点で優れています。開発の現場で実装をする機会自体は少ないですが、その仕組みや線形探索と比較したメリットを知っていることは重要です。
